## Node Structure

Each of the 41 nodes includes:
- **Metadata:** Code, title, year, strand, cascade path
- **Learning Outcomes:** Aligned with NaCCA curriculum
- **Indicators:** Object-based with full NaCCA indicator text
- **Prerequisites and Downstream Support:** Cascade relationships
- **Diagnostic Prompt:** Ghana-contextualized assessment examples
- **Assessment Protocol:** Rubric, time, materials, scoring, proficiency levels
- **Error Patterns:** 5-7 patterns with frequency, behaviors, diagnostic value
- **Misconceptions:** 3-5 with codes (MC-E01 to MC-E05) from evidence_base.json
- **Difficulty Estimate:** 1-7 scale with WASSCE data
- **Proficiency Benchmark:** Mastery description and observable indicators
- **Intervention Protocol:** 4-6 sessions with Ghana classroom adaptations
- **Ghana Context:** Materials, large class (45-60), L2 considerations, WASSCE relevance
- **Research Notes:** Evidence base citations, cascade impact, effectiveness data

## Ghana Context Features

### Oral Communication (NEW nodes)
- **L1 Interference:** /θ/→/t/, /v/→/f/ substitutions (85-95% of students)
- **Pronunciation Gaps:** Vowel length, diphthongs, consonant clusters
- **Silent Letters:** Major challenge (comb, listen, subtle)
- **Listening Comprehension:** Note-taking strategies, main idea vs details
- **Register and Pragmatics:** Formal vs informal, speech acts

### Literature (NEW nodes)
- **Year 1:** African poetry focus (cultural familiarity)
- **Year 2:** Non-African poetry (cross-cultural analysis)
- **Literary Devices:** Metaphor, simile, imagery, theme
- **WASSCE Relevance:** Literature paper tested

### Assessment Alignment
- **WASSCE Oral Test:** 20% of English grade
  - Part A: Reading aloud (pronunciation)
  - Part B: Picture description
  - Part C: Conversation (register, appropriacy)
- **Large Class Adaptation:** 45-60 students per class
- **Minimal Resources:** Chalkboard, textbooks, exercise books only
