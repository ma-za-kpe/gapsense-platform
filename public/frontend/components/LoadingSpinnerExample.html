LoadingSpinner Component Examples

Mobile-optimized loading spinners with hardware-accelerated animations

1. Global Overlay Spinner

Fullscreen loading spinner with overlay (use for page-level loading)

2. Inline Spinners (Small, Medium, Large)

Inline spinners for use within containers

3. Skeleton Loaders

Shimmer effect for content placeholders

4. Programmatic Usage (JavaScript)

// Import the LoadingSpinner
import { showLoading, hideLoading, updateLoadingMessage } from './components/LoadingSpinner.js';

// Show loading spinner
showLoading('Uploading image...');

// Update message
updateLoadingMessage('Analyzing exercise book...');

// Hide loading spinner
hideLoading();

// Or create custom instance
import LoadingSpinner from './components/LoadingSpinner.js';

const spinner = new LoadingSpinner({
  type: 'dots',          // 'spinner', 'dots', or 'pulse'
  size: 'medium',        // 'small', 'medium', or 'large'
  color: '#25D366',      // CSS color
  message: 'Loading...',
  overlay: false         // true for fullscreen overlay
});

spinner.show();
spinner.updateMessage('Processing...');
spinner.hide();
