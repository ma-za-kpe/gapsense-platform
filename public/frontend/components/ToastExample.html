Toast Notification Component - Mobile-Optimized

Lightweight notification system for mobile devices with swipe-to-dismiss

1. Basic Toast Types

Four variants: success, error, warning, and info

2. Toast Positions

Top or bottom positioning (optimized for mobile safe areas)

3. Auto-Dismiss Duration

Customize how long toasts stay visible (or make them persistent)

4. Multiple Toasts (Stacking)

Show multiple toasts at once - automatically manages max 3 toasts

5. Touch Gestures (Mobile)

Try on mobile: Swipe left/right to dismiss, or tap the × button

  • Swipe left or right > 100px to dismiss
  • Tap × button to dismiss
  • Auto-dismiss after duration
  • Opacity fades during swipe

6. Programmatic Usage (JavaScript)

// Import the Toast module
import { successToast, errorToast, warningToast, infoToast, showToast } from './components/Toast.js';

// Show basic toasts
successToast('Exercise book uploaded successfully!');
errorToast('Failed to analyze image. Please try again.');
warningToast('Network connection is unstable...');
infoToast('Processing your image...');

// Custom options
showToast('Custom toast message', {
  type: 'success',       // 'success', 'error', 'warning', 'info'
  position: 'top',       // 'top' or 'bottom'
  duration: 5000,        // milliseconds (0 = no auto-dismiss)
  dismissible: true,     // show × button
  icon: '🎉'             // custom icon
});

// Advanced: Get toast instance for control
const toast = successToast('Uploading...', { duration: 0 });
// Later: toast.hide();

// Clear all toasts
import { clearAllToasts } from './components/Toast.js';
clearAllToasts();

8. Mobile Optimization Features

  • Hardware-accelerated - Smooth 60fps animations
  • Touch-friendly - 44px minimum touch targets
  • Safe area support - Respects iPhone X+ notch/home indicator
  • Swipe gestures - Natural mobile dismiss interaction
  • Auto-stacking - Max 3 toasts, removes oldest automatically
  • Reduced motion - Respects prefers-reduced-motion
  • Dark mode - Automatic theme switching
  • High contrast - Accessible for all users
  • ARIA labels - Screen reader support
  • Lightweight - ~3KB minified + gzipped
