📐 Responsive Grid System

Mobile-first, flexible grid layout system for GapSense platform

XS (<640px) Mobile (640px+) Tablet (768px+) Desktop (1024px+)

✨ Key Features

  • Mobile-First - Base styles for mobile, progressively enhanced for larger screens
  • Auto-Fit Grids - Automatically adjust columns based on available space
  • Responsive Breakpoints - sm (640px), md (768px), lg (1024px), xl (1200px)
  • Flexible Gaps - sm (8px), md (16px), lg (24px), xl (32px)
  • Spanning Utilities - Span multiple columns at any breakpoint
  • Alignment Control - Align and justify grid items
  • Stack Layout - Vertical stacking with configurable gaps
  • Accessibility - Focus states, reduced motion support
  • Performance - Layout containment, hardware acceleration
  • Print Optimized - Responsive layout for printing
