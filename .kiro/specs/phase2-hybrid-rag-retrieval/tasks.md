# Implementation Plan: Phase 2 ÔÇö Hybrid RAG Retrieval

## Overview

Replace the brute-force `_build_curriculum_graph` in `ImageAnalysisOrchestrator` with a hybrid retrieval pipeline: pgvector cosine similarity search + recursive CTE prerequisite walk. Implementation follows dependency order: migration Ôćĺ EmbeddingService Ôćĺ embedding job Ôćĺ context extensions Ôćĺ orchestrator methods Ôćĺ WorkerService wiring Ôćĺ template update.
